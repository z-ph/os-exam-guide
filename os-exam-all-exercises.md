## 考试重点题号（必会）

### 计算题（10分/题，共40分）

1. **Ch2 PV操作**：水果盘问题 或 单缓冲区采集计算 或 三进程文件打印
2. **Ch3 调度算法**：FCFS/SJF/高响应比 的周转时间、带权周转时间计算
3. **Ch4 地址变换**：逻辑地址→物理地址（分页/分段）、TLB有效访问时间
4. **Ch5 页面置换**：OPT/FIFO/LRU 缺页次数计算、缺页率、有效访问时间

### 概念分析题（可能考这些）

- 进程与程序的区别
- 死锁四个必要条件 + 预防方法
- 四种I/O控制方式对比
- FAT12/16/32的区别与容量计算
- 混合索引（直接/一次间接/二次间接/三次间接）
- 动态重定位 vs 静态重定位
- 紧凑的概念与作用
